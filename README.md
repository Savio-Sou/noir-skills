# Noir Skills

Prompts in markdown format describing common patterns and best practices for agentic workflows with Noir.

## Installation

The simplest way is usually to:

1. Clone this repository
```bash
git clone https://github.com/Savio-Sou/noir-skills.git
```
2. Copy contents in noir-skills' `skill` folder into your project's / your AI tooling's skill folder.
